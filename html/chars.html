- Caracteristicas -

Definiciones

HTML

  • Es un lenguaje que se puede entender fácilmente y se puede modificar.
  • Ofrece compatibilidad con todos los navegadores web.
  • HTML crea documentos estructurados al denotar semántica estructural para texto como encabezados, listas, enlaces, citas, etc.

CSS

  • Una característica importante de CSS sería las reglas de estilo que son interpretadas por el navegador del cliente y aplicadas a varios elementos de su documento.
  • Si tiene la intención de realizar algún cambio global, simplemente puede cambiar el estilo y podrá ver que todos los demás elementos en todas las demás páginas web se actualizan automáticamente.
  • CSS se considera una hoja de estilo muy conveniente y fácil de leer. Esto significa que los motores de búsqueda no tienen que esforzarse mucho para intentar leer el texto.

XHTML

  • Los documentos XHTML están bien formados y, por lo tanto, se pueden analizar mediante analizadores XML estándar, a diferencia de HTML, que requiere un analizador específico de HTML.
  • Al usar espacios de nombres, los documentos XHTML podrían proporcionar extensibilidad al incluir fragmentos de otros lenguajes basados en XML, como Scalable Vector Graphics y MathML.
  • Es una familia de lenguajes de marcado XML que reflejan o amplían versiones del HTML ampliamente utilizado.

XML

  • Es un lenguaje de marcado basado en texto derivado del lenguaje de marcado estándar generalizado (SGML).
  • Las etiquetas XML identifican los datos y se utilizan para almacenar y organizar los datos, en lugar de especificar cómo mostrarlos como etiquetas HTML, que se utilizan para mostrar los datos.
  • XML le permite crear sus propias etiquetas autodescriptivas, o lenguaje, que se adapte a su aplicación.
  • XML le permite almacenar los datos independientemente de cómo se presenten.
  • XML fue desarrollado por una organización llamada World Wide Web Consortium (W3C) y está disponible como estándar abierto.

Markdown

  • Markdown es un lenguaje de marcado ligero con sintaxis de formato de texto plano.
  • Hay editores especialmente diseñados que previsualizan los archivos con estilos, que están disponibles para todas las plataformas principales.
  • Markdown es un lenguaje de formato como HTML que puede utilizar para especificar la apariencia final del texto.

XPath

  • XPath es un lenguaje de consulta para seleccionar nodos de un documento XML. Además, XPath se puede utilizar para calcular valores a partir del contenido de un documento XML.
  • El lenguaje XPath se basa en una representación de árbol del documento XML y proporciona la capacidad de navegar por el árbol, seleccionando nodos según una variedad de criterios.
  • XPath se puede utilizar para documentos HTML y XML para encontrar la ubicación de cualquier elemento en una página web utilizando la estructura DOM de HTML.

XQuery

  • XQuery es compatible con las principales bases de datos.
  • XQuery es el lenguaje para consultar datos XML.
  • XQuery está reemplazando programas complejos de Java o C ++ con unas pocas líneas de código.
  • Utiliza expressiones XPath para acceder a determinadas partes del documento XML. Añade, además, expresiones similares a las usadas en SQL, conocidas como expresiones FLWOR
  • Las expresiones FLWOR toman su nombre de los 5 tipos de sentencias de las que pueden estar compuestas: FOR, LET, WHERE, ORDER BY, y RETURN
